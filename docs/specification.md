## 기획
- 목적 : 기부 문화를 활성화한다.
- 판매자는 물품을 기부할 수 있다.
- 구매자는 물품을 경매를 통해 구매한다.

판매자, 구매자 모두 손해봐도 괜찮다는 마인드 갖기 "어짜피 기부하는 거니까~"

## 기능 명세서
- 기부 물품
  - 이름
  - 설명
  - 경매 시작가
  - 상태 (경매 시작 전, 경매 중, 낙찰 완료, 입금 완료, 배송완료)
- 경매장
  - 시작 날짜
  - 종료 날짜
  - 채팅
  - 경매 진행
- 판매자
  - 물품 등록
  - 물품 배송
  - 낙찰 알림 받기
- 구매자
  - 경매장 입장
  - 가격 제시
  - 입찰
  - 금액 송금
