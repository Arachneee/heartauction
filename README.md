## 기획
- 목적 : 기부 문화를 활성화한다.
- Flow
    - 판매자는 물품을 기부 등록한다.
    - 구매자는 물품을 경매를 통해 낙찰받는다.
    - 구매자는 낙찰된 금액을 송금한다.
    - 판매자가 물품을 배송한다.
    - 낙찰된 돈을 기부한다.

판매자, 구매자 모두 손해봐도 괜찮다는 마인드 갖기 "어짜피 기부하는 거니까~"
